Programkod

I detta kapitel finns exempel på programkod. Programmet är ett enkelt testprogram skrivet i Java.

package helloworldapp;

/**
 * This class implements an application that simply
 * prints "Hello World!" to standard output.
 */
public class HelloWorldApp {

    public static void main(String[] args) {
        System.out.println("Hello World!");
    }

}
